# 安装

npm install qzd-js
or
yanr add qzd-js

支持页面引入

<script src="https://qzd-web-group-oss.qizhidao.com/wx/cdn/qzdjs/0.1.2-alpha.27/qzd.min.js"></script>

# 使用方法

import QzdJs from "qzd-js"

async function login () {
   const { ticket } = await Qzdjs.login()
}

# 接口文档的说明

** name** 字段为方法描述

** since** 字段为APP最低支持版本

** description ** 字段为特殊说明

** alias** 字段为对应的原始方法名(原始方法名有一些存在不规范问题)

# setNavButton

▸ setNavButton(data): Promise<unknown>

name 设置导航按钮

description H5配置原生导航栏右侧按钮, 右侧按钮列表（目前只接受一个，配置多个也默认去第一个，后续决定是否实现多个）, 原生会在h5容器右侧展示h5配置的按钮，当用户发生按钮操作时，原生会通过onEventChangeCallback回调通知h5操作事件, 对调事件需将按钮tag作为参数传入，h5根据tag识别自己对应tag按钮：形如：onEventChangeCallback({tag:1,status:1}) 此接口目前仅使用在购物车页面

# Parameters

Name Type
data SetNavButtonOpt

# Returns

Promise<unknown>
